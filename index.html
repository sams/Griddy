griddy

Griddy a simple script that overlays a defined grid to select page elements

Simple, effective and above all usable

Griddy by sams

this page is a bit unhinged :)

Griddy is a small JQuery plugin thats creates a simple, customizable overlay on top of any element. It can also calculate row heights and column widths automatically based on number of rows/columns and gutter space.

Traditional Markup inclusion of Griddy

Insert Griddy into you head

<script src="http://sams.github.com/Griddy/griddy.min.js" type="text/javascript"></script>

jQuery on ready setup of griddy defining a grid for selected element

$('.element').griddy({
			rows: 10,
			rowheight: 0,
			rowgutter: 20,
			columns: 4,
			columnheight: 0,
			columngutter: 20,
			color: '#ccc', 
			opacity: 30
		});
